What We Found

Using Xquery in eXide, we first counted how many pronouns of all the pronouns we marked were archaic and how many were modern. We used this Xquery:

                xquery version "3.0";
declare namespace tei="http://www.tei-c.org/ns/1.0";
let $docTexts := doc('/course/iberian/data/songTextsThirdTry.xml')/tei:TEI
let $docFeat := doc('/course/iberian/aux/features.xml')/tei:TEI
let $modern := $docTexts//tei:w[substring(@ana,2) = $docFeat//tei:fs[tokenize(@feats/string(),'\s+') = '#archaic']/@xml:id]
 return $modern/string()

Then we used another query to filter the results by period:

                xquery version "3.0";
declare namespace tei="http://www.tei-c.org/ns/1.0";
let $docTexts := doc('/course/iberian/data/songTextsThirdTry.xml')/tei:TEI
let $docFeat := doc('/course/iberian/aux/features.xml')/tei:TEI
let $period := $docTexts//tei:div[descendant::tei:affiliation/@corresp='1']
let $modern := $period//tei:w[substring(@ana,2) = $docFeat//tei:fs[tokenize(@feats/string(),'\s+') = '#archaic']/@xml:id]
 return $modern
